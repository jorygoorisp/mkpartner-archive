# mkpartner-archive

本仓库用于归档和发布多个独立 HTML 页面。这些页面为不同场景下可离线或在线查看的静态内容，不针对任何特定域名或网站。

## 项目简介

mkpartner-archive 是一个静态 HTML 页面集合仓库。页面内容以独立文件形式存放，便于浏览、分享或嵌入其他项目。本仓库不提供任何服务承诺，仅作为归档用途。

## 目录说明

- `pages/` —— 存放所有归档的 HTML 页面文件
- `assets/` —— 页面引用的样式、脚本、图片等静态资源
- `index.html` —— 仓库首页，可快速浏览所有页面列表

## 页面归档说明

每个 HTML 页面均保持独立、自包含（或通过相对路径引用资源）。页面内容可能涉及不同主题，但均不指向或推广任何具体商业服务。归档仅用于技术演示、学习交流或历史记录。

## 维护说明

- 本仓库由维护者不定期更新，不保证更新频率
- 页面内容可能根据实际情况进行调整或移除
- 欢迎通过 Issue 提交建议，但请勿要求添加特定商业或推广内容

## 使用方式

- 直接访问 GitHub Pages 部署链接（如有启用）查看页面
- 克隆仓库后，在本地浏览器打开 `index.html` 或 `pages/` 下任意文件
